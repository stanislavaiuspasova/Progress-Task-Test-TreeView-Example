TreeView / Checkboxes Test Scenarios

Expand nodes

 Expand the parent node and verify that all children nodes are displayed

Collapse nodes

 Collapse the parent node and verify that all children nodes are collapsed

Check nodes

 Check the parent node and verify that all children nodes are checked as well

 Check all children nodes and verify that the parent is checked

 Check a child node and verify that the parent is selected

Uncheck nodes

 Unheck the parent node and verify that all children nodes are unchecked as well

 Uncheck all children nodes and verify that the parent is unchecked

 Uncheck a child node and verify that the parent is not selected

Verify the status of the nodes

 Check some nodes and verify that the status shows their correct IDs

 Make sure no nodes are checked and verify that the status displays no IDs

Verify the appearance of the nodes

 Verify that each node displays the correct icon beside the text

Verify the keyboard navigation

 Verify that the supported keyboard shortcuts navigate through the TreeView

Verify accessibility

 Verify that the TreeView is accessible by screen readers and provides full WAI-ARIA support
